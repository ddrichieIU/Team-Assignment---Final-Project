About Movie Vault

Built for movie discovery.

Movie Vault is a movie database website inspired by IMDb and styled with a dark, modern theater feel. Users can browse trending titles, search for movies, view detailed information, and save favorites to a personal watchlist.

This website is designed to make exploring movies simple and enjoyable. Users can browse trending titles, search for movies they’re interested in, view detailed information, and build a personal watchlist of their favorites all in one place. The goal is to create an experience that feels smooth, organized, and easy to use for anyone who loves movies.

Core Features

  • Trending movie browsing
  • Search by title
  • Movie details page
  • Watchlist / favorites

Tech Used

  • HTML
  • CSS
  • JavaScript
  • TMDB API
  • Firebase
